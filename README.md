# patrickroos.io

Minimal personal site built with Astro.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content

- Home/About: `src/pages/index.astro`
- Posts: `src/content/posts/*.md`
- Publications: `src/data/publications.json`

## Deployment

Deploy on Cloudflare Pages using:

- Build command: `npm run build`
- Output directory: `dist`
